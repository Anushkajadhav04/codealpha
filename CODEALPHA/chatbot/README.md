# 🤖 GUI Chatbot (Python Tkinter)

## 📌 Description
This project is a GUI-based chatbot built using Python's Tkinter library. It provides interactive responses and a user-friendly chat interface.

---

## 🚀 Features
- Graphical User Interface (GUI)
- Chat window with scroll feature
- User input box with Enter key support
- Smart responses (time, date, GK, calculations)
- Styled chat messages

---

## 🧠 Concepts Used
- Tkinter (GUI development)
- Functions
- Conditional statements
- Event handling
- `datetime` module

---

## ▶️ How to Run
1. Save the file as `gui_chatbot.py`
2. Run:
   ```bash
   python gui_chatbot.py
