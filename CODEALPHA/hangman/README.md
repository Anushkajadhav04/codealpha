# 🎮 Hangman Game (Python)

## 📌 Description
This is a simple text-based Hangman game implemented in Python. The player has to guess a randomly selected word one letter at a time within a limited number of attempts.

---

## 🚀 Features
- Random word selection
- Maximum 6 incorrect guesses allowed
- Displays guessed letters and remaining attempts
- Win/Lose conditions
- User-friendly console interface

---

## 🧠 Concepts Used
- Python loops (`while`)
- Conditional statements (`if-else`)
- Lists and strings
- `random` module

---

## ▶️ How to Run
1. Install Python
2. Save the file as `hangman.py`
3. Run:
   ```bash
   python hangman.py
