# 📊 Stock Portfolio Tracker (Python)

## 📌 Description
This project is a simple stock portfolio tracker that calculates total investment based on user input and predefined stock prices.

---

## 🚀 Features
- User inputs stock names and quantities
- Predefined stock price dictionary
- Calculates total investment
- Saves output to a file (`portfolio.txt`)

---

## 🧠 Concepts Used
- Dictionaries
- Loops
- Input/Output
- File Handling

---

## ▶️ How to Run
1. Save the file as `stock_tracker.py`
2. Run:
   ```bash
   python stock_tracker.py
