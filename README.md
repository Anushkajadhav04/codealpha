# CodeAlpha Python Internship Projects

This repository contains Python projects completed as part of the **CodeAlpha Python Programming Internship**.
These projects demonstrate core Python concepts including logic building, file handling, and GUI development.

---

# Projects Included

## 🎮 1. Hangman Game

### 📌 Description

A simple text-based Hangman game where the player guesses a randomly selected word one letter at a time within limited attempts.

### 🚀 Features

* Random word selection
* Maximum 6 incorrect guesses
* Displays progress using `_ _ _` format
* Win/Lose conditions

### 🧠 Concepts Used

* `random` module
* Loops (`while`)
* Conditional statements
* Strings & lists

### ▶️ How to Run

```bash
python hangman.py
```

## 📊 2. Stock Portfolio Tracker

### 📌 Description

A Python program that calculates total investment based on user input and predefined stock prices.

### 🚀 Features

* Input stock name and quantity
* Predefined stock price dictionary
* Calculates total investment
* Saves results to `portfolio.txt`

### 🧠 Concepts Used

* Dictionaries
* Input/Output
* Loops
* File handling

### ▶️ How to Run

```bash
python stock_tracker.py
```


## 🤖 3. GUI Chatbot (Tkinter)

### 📌 Description

A smart chatbot built using Python Tkinter with a graphical user interface. It can respond to user queries, show time/date, answer basic general knowledge questions, and perform calculations.

### 🚀 Features

* GUI-based chatbot
* Chat window with scroll
* Enter key support
* Smart responses (time, date, GK, calculator)
* Styled chat interface

### 🧠 Concepts Used

* Tkinter (GUI development)
* Functions
* Event handling
* `datetime` module

### ▶️ How to Run

```bash
python gui_chatbot.py
```
